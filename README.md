# Inventory Management System

This is a VB.NET Windows Forms inventory management system created for learning purposes. The project is used to practise desktop application development, CRUD operations, database connection, stock management, reporting, and user interface design before moving on to larger and more advanced software projects.

## Project Overview

The Inventory Management System is a desktop application designed to help manage basic inventory records. It includes modules for products, categories, suppliers, stock, reporting, login, and dashboard navigation.

This project is mainly built as a learning project to improve my understanding of VB.NET, Windows Forms, SQL Server LocalDB, and basic software development structure.

## Features

- Login form
- Show/hide password function
- Dashboard form
- Product management
- Category management
- Supplier management
- Stock management
- Reporting module
- Load categories and suppliers into ComboBox
- Add, update, delete, and clear form data
- SQL Server LocalDB database connection
- Organised project structure using Forms, Helpers, Models, and Data folders

## Technologies Used

- VB.NET
- Windows Forms
- SQL Server LocalDB
- Visual Studio
- Git
